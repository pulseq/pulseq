March 2026: new Pulseq bug fix interpreter release for Siemens

April 2026

We have made a substantial progress in resolving the infamous table undocked problem (scanner controller crash and restart) and release a patch under a version number v1.5.2p1. This version is a pure bug-fix release and does not contain any new features. In addition to fixing (hopefully all manifestations of the table undocked problem, but at least their substantial part), there is a fix included to setting the SET counter limit, as well as a fix for importing stored older Pulseq protocols leading to an
incorrect setting of the "Minimum segment length" parameter on the sequence-special card.
